Item 144 : Fièvre aigüe

Fièvre : 

  •  ≥ 38o le matin, 38.3o le soir
  • aigüe si  < 5 jours
  • prolongée si  > 20 jours

⚠ fièvre ≠ infection

Signes de gravité :

  • neuro: angoisse, agitation, confuion, troubles du comportement, coma
  • cardiovasc : FC > 120/min, TA systol < 90 mmHg, PAM < 65 mmHg
  • cutané : purpura, extrémités froides et cyanosées, marbrures
  • respiratoire : polypnée > 24/min, tirage, balancement thoraco-abdominal, polypnée superficielle, SaO2 < 90%

Peut décompenser une comorbidité :

  • neurologique
  • 37 + n∘ =>  + 400 ⋅ n mL/j pertes hydriques
  • 37 + n∘ =>  + 10 ⋅ n battements/min pour FR et FC
  • augmentations des besoins en oxygène

Terrain à risque : femme enceinte, immunodépression

⚠ ATB sans diagnostic seulement pour sepsis/grave, choc septique, neutropénie ( < 500PNN/mm3), asplénie, pupura fulminans

Traitement sympotmatique : antipyrétiques seulement si fièvre mal tolérée/terrain particulier => paracétamol (aspirine, AINS non recommandés)

Signes de gravité :

  • purpura extensif
  • trouble de la conscience (Glasgow <= 8)
  • signes de focalisation neuro
  • signes de souffrance du tronc cérébral
  • état de mal convuslif
  • instabilité hémodynamique

Contre-indication à la PL :

  • anomalie de l’hémostase [PL dès stabilité]
  • instabilité hémodynamique [PL dès stabilité]

  • crise convulsive récente
  • risque d’engagement cérébral (localisation neuro, trouble vigilance + Glasgow <= 11)

Item 149 : Endocardite infectieuse

Agents infectieux :

  • Staphylococcus aureus
  • Streptocoques oreaux
  • Streptococcus gallolyticus
  • Entérocoques

Diagnostic =

  • fièvre + souffle cardiaque (nouveau/modifié)
  • agent infectieux identifié
  • anomalie intracardiaque

⚠ Fièvre inexpliquée chez valvulopathie = EI par défaut

⚠ Signe neuro. fébrile -> chercher EI (ascult + hémoc.)

Cardiopathies à haut risque :

  • prothèse valvulaire
  • cardiopathies congénitales cyanogène + shunt persistant + dérivation chir.
  • ATCD d’EI

ATB adaptée :

  • Staph. aureus : penicilline M IV
  • Strept. oraux : amoxicilline IV, ceftriaxone IV
  • Strept. gallolyticus : amoxicilline IV, ceftriaxone IV
  • Enteroccocus spp. : amoxicilline + gentamicine IV ou amoxicilline + ceftriaxone (Si allergie/résistance : glycopeptide)

Critèse de Duke : certitude = 2 majeurs / 1 majeurs + 3 mineurs / 5 mineurs

Virales (papulo-)maculeuses

Morbiliforme

Rubéole :

  • contagieuse, contamination aérienne
  • immunisante, bénigne sauf grossesse
  • phase d’état : éruption morbiliforme (inconstante, souvent discrète 20%)
  • complications : polyarthrite, thrombopénie, encéphalite
  • embryon/foetus : risque du sd malformatif de Gregg
  • diagnostic : clinique trompeuse, sérologie (IgM), NFS
  • traitement préventif : vaccin vivant atténué ROR (12M, 16-18M)

Rougeole

  • très contagieux, contamination aérienne
  • diagnostic : clinique++, NFS, PCR, sérologie
    • catarrhe occulo-naso-bronchique fébrile
    • signe de Köplick, exanthème maculopapuleux
  • complications :
    • respiratoires : surinfections bactériennes (otites, laryngites, bronchites), pneumonies bactériennes/viral
    • encéphalites : encéphalomyélite aigüe post-éruptive, encéphalite aigüe progressive, panencéphalite sclérosante subaigüe (PESS)
  • traitement préventif : vaccin vivant atténué ROR (12M, 16-18M)

Mégalérythème épidémique

  • contagion respiratoire
  • clinique : rash maculopapuleux (joue -> tronc -> extrémités), arthralgies
  • diagnostique : sérologie, PCR

Exanthème subit :(HHV-6)?

  • enfant 6 mois - 4 ans
  • clinique :
    • fièvre 3-4 jours
    • exanthème maculo-papuleux rose pâle + enanthème voile du palais (éruption 24h)
  • traitement symptomatique
  • complications : celles de la fièvre

Autres exanthèmes :

  • adénovirus
  • zika
  • VIH

Scarlatiniforme

Scarlatine

  • clinique : 
    • début brutal (fièvre, céphalées, maux gorge, vomissements)

Syndrome de Kawasaki

  • vascularite systémique aigüe
  • clinique :
    • fièvre prolongée, exanthème, glossie
    • adénopathies cervicales volumineuses
    • desquamation en doigts de gants
  • jeunes enfants
  • traitement : Ig IV 2j puis aspirine 14j

Sd du choc toxique

  • Tampon périodique > 8h
  • Clinique : fièvre, état de choc, érythrodermie généralisée, desquamation tardive
  • traitement : clindamycine + gentamicine IV

Item 161 : Oreillons

Bénigne, contagieuse à réservoir humain Diagnostic principalement clinique :

  • incubation 19j
  • invasion 24-48h
  • état : parotidite ourliennes (70%) unilatérale puis bilatérale

Guérison spontanée 8-10j Autres formes :

  • neuroméningé++ : méningite lymphocytaire (pas de séquelles), encéphalite (décès 1-5%, surdité possible)
  • orchite/épidymite ourlienne, atrophie testiculaire 50%
  • pancréatie ourlienne : rare, guérison spontanée sans séquelle

Traitement symptomatique
Prévention : vaccin ROR

Varicelle

90% chez l’enfant, plus grave chez l’adulte Diagnostique clinique

  • incubation (14j)
  • état : (maculo-papule) -> vésicules (prurit+++) ->_érosion + pseudo-ombilication -> croûtes (J4) -> cicatrisation (J10)
  • spontanément favorable (10-15j)

Complications :

  • surinfection bactérienne fréquente (grattage)
  • terrains, état infectieux sévère, respiratoie neuro, purpura thrombopénique aigu

Traitement :

  • symptomatique (pas d’aspirine, ni AINS ⚠)
  • ATB si surinfection cutanée, antiviral si formes graves/compliquées
  • pas d’éviction scolaire
  • vaccination (jeunes non immuns)

Zona

Réactivation du VZV. Gravité : douleurs post-zostériennes, immunodépression, localisations

Diagnostic : clinique

  • prodrome
  • état : éruption (érythémateux roses vifs -> vésicules en bouquets -> érosives -> croûtes -> cicatrices dépigmentées (rosées -> blanchâtres) + fébricule
  • poussées sur 2-3 semaines, douleurs plusieurs mois

Complications : principalement DPZ (rarement neurologique)

Traitement

  • local : symptomatique, douleur (antalgique phase aigüe)
  • ATB si surinfection cutanée, antiviral en prévention DPZ (valaciclovir per os 7 jours)
  • prévention vaccination (entre 65 et 74 ans)

Complications

HSV1, 2 : - nouveau-né (périnatale++) : mortalité très élevée, séquelle très lourdes - femme enceinte : maladie chez nouveau-né - atopique : Kaposi-Juliusberg VZV
